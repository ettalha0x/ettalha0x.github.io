About This Project

so_long was inspired by classic maze and puzzle games. The challenge of navigating complex environments while collecting items and avoiding traps has always fascinated me. I chose this project as part of the 42 curriculum to dive deeper into game development, understand rendering techniques, and work closely with low-level programming in C.
